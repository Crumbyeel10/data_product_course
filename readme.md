# Data Product Course (Capstone: Ventas + Clientes)

## Objetivo
Construir un data product end-to-end: ingesta (raw/bronze), limpieza (silver) y datasets listos para negocio (gold),
con validaciones de calidad y documentación.

## Estructura
- data/raw: datos crudos (bronze)
- data/processed: salidas silver/gold
- notebooks: exploración y pruebas
- src: código del pipeline
- tests: validaciones y pruebas
- docs: documentación (métricas, medallion, runbook, backlog)
